# Output Folder

This folder stores generated Instagram story images.

## Purpose

- Store daily weather forecast images (1080x1920px)
- Ready for upload or email to social media manager

## File Naming Convention

- Format: `weather_story_YYYY-MM-DD.jpg`
- Example: `weather_story_2025-10-15.jpg`

## File Specifications (Phase 2)

- Size: 1080x1920 pixels (Instagram story format)
- Format: JPG
- Content: 15 Israeli cities with weather forecast
- Layout: North to South by latitude

- **Phase 2:** Image generation will be implemented
